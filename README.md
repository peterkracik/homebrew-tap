# homebrew-tap

Homebrew formulae for [@peterkracik](https://github.com/peterkracik)'s tools.

## Usage

```bash
brew tap peterkracik/tap
```

## Formulae

### devsecrets

Terminal utility for managing local development secrets across projects and environments.

```bash
brew install peterkracik/tap/devsecrets
```

Source: [peterkracik/dev-secrets](https://github.com/peterkracik/dev-secrets)
